Useful Functions¶

Fields¶

For easy field creation.

Function Description
viscid.arrays2field() Field from arrays for data and coordinates
viscid.dat2field() Field from data array
viscid.empty() Make an empty field from coordinate arrays
viscid.zeros() Make an field of 0s from coordinate arrays
viscid.ones() Make an field of 1s from coordinate arrays
viscid.empty_like() Make an empty field like another
viscid.zeros_like() Make a field of 1s like another
viscid.ones_like() Make a field of 0s like another

Calculations¶

Streamlines and Interpolation¶

Class Description
viscid.calc_streamlines() Calculate streamlines
viscid.interp_trilin() Trilinear interpolation
viscid.interp_nearest() Nearest neighbor interpolation
viscid.Point Collection of hand picked points
viscid.Line A line between 2 points
viscid.Plane A plane difined be an origin and a normal
viscid.Volume A Volume of points on a uniform cartesian grid
viscid.Sphere Points on the surface of a sphere
viscid.SphericalCap A cap of points around the pole of a sphere
viscid.Circle Just a circle
viscid.SphericalPatch A rectangular patch on the surface of a sphere

Math¶

These functions will by accelerated by Numexpr if it is installed.

Function Description
viscid.add() Add two fields
viscid.diff() Subtract a field from another
viscid.mul() Multiply two fields
viscid.relative_diff() Divide the difference by the magnitude
viscid.abs_diff() Absolute value of the difference
viscid.abs_val() Absolute value
viscid.abs_max() Max of the absolute value
viscid.abs_min() Min of the absolute value
viscid.magnitude() Magnitude of a viscid.field.VectorField
viscid.dot() Dot product of two viscid.field.VectorField
viscid.cross() Cross product of two viscid.field.VectorField
viscid.div() Divergence of a viscid.field.VectorField
viscid.curl() Curl of a viscid.field.VectorField
viscid.project() Project one viscid.field.VectorField onto another
viscid.integrate_along_lines() Integrate a field along streamlines
viscid.calc_psi() Calculate a 2D flux function
viscid.calc_beta() Calculate plasma beta

Plotting¶

General Matplotlib Functions¶

These functions wrap Matplotlib with useful boilerplate type hacks.

Function Description
viscid.plot.mpl.plot.auto_adjust_subplots() Use Matplotlib’s tight layout with some necessary hacks

2D Matplotlib Plots¶

These functions wrap Matplotlib with useful boilerplate type hacks.

Function Description
viscid.plot.mpl.plot() Meta function for plotting viscid.field.Field objects. This one will automatically delegate to viscid.plot.mpl.plot1d_field(), viscid.plot.mpl.plot2d_field(), or viscid.plot.mpl.plot2d_mapfield().
viscid.plot.mpl.plot1d_field() Line plots of a 1D field.
viscid.plot.mpl.plot2d_field() Colored plots (pcolormesh, contour, contourf) of 2D fields
viscid.plot.mpl.plot2d_mapfield() Plots on the surface of a sphere (like ionosphere plots)
viscid.plot.mpl.plot2d_lines() Plot a list of colored lines parallel-projected into 2D
viscid.plot.mpl.plot2d_quiver() Plot a viscid.field.VectorField using Matplotlib’s quivers.
viscid.plot.mpl.streamplot() Plot a viscid.field.VectorField using Matplotlib’s streamplot.
viscid.plot.mpl.plot_earth() Plot an Earth with black for nightside and white for dayside

3D Matplotlib Plots¶

These functions wrap Matplotlib in 3D with useful boilerplate type hacks.

Function Description
viscid.plot.mpl.plot3d_lines() Plot a list of colored lines on 3D axes
viscid.plot.mpl.scatter_3d() Plot a glyphs on 3D axes

3D Mayavi Plots¶

These functions wrap Mayavi with useful boilerplate type hacks.

Function Description
viscid.plot.mvi.add_source() Given a VTKDataSource, add it to a figure
viscid.plot.mvi.add_lines() Given a list of lines, add them to a figure as a data source
viscid.plot.mvi.add_lines() Given a viscid.field.Field, add it to a figure as a data source
viscid.plot.mvi.plot_mesh() Plot viscid.Seed objects as a mesh
viscid.plot.mvi.plot_lines() Plot colored lines in 3D
viscid.plot.mvi.plot_ionosphere() Plot an ionospheric Field in 3D
viscid.plot.mvi.insert_filter() Insert a filter above a module_manager.
viscid.plot.mvi.plot_blue_marble() Plot an Earth using the blue marble NASA image
viscid.plot.mvi.plot_earth_3d() Plot an Earth with black for nightside and white for dayside
viscid.plot.mvi.clf() Uses some hacks to clear a figure and make sure memory is freed
viscid.plot.mvi.resize() Uses some hacks to resize a figure
